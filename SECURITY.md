# Polityka bezpieczeństwa / Security Policy

## Wspierane wersje / Supported Versions

Projekt Site_proj jest aktywnie rozwijany. Obecnie poprawki związane z bezpieczeństwem otrzymuje wyłącznie najnowsza wersja opublikowana jako release.

| Wersja / Version | Wspierana / Supported    |
| --------------- | ------------------------ |
| 2.0v            | ✅ Tak / Yes             |
| < 2.0v          | ❌ Nie / No              |

Poprawki bezpieczeństwa są dostarczane wyłącznie dla wersji 2.0v. Starsze wydania nie otrzymują wsparcia w zakresie podatności.

## Zgłaszanie podatności / Reporting a Vulnerability

Jeśli znalazłeś/aś lukę bezpieczeństwa w projekcie Site_proj:

1. Skontaktuj się przez e-mail: **szymon.przybysz2003@gmail.com** lub utwórz zgłoszenie w zakładce [Issues](https://github.com/AdasRakieta/Site_proj/issues) (oznacz jako "security").
2. Opisz problem, kroki do odtworzenia oraz potencjalne skutki.
3. Wstępna odpowiedź następuje zwykle w ciągu **5 dni roboczych**.
4. Po weryfikacji i usunięciu podatności informacja o problemie oraz naprawa zostanie publicznie udostępniona, z podziękowaniem dla osoby zgłaszającej.
5. Nie ujawniaj szczegółów podatności publicznie, dopóki nie zostanie naprawiona.

## Obsługa zgłoszeń i komunikacja

Projekt utrzymuje obsługę zgłoszeń bezpieczeństwa w miarę możliwości przez autora (AdasRakieta) oraz głównych kontrybutorów. W przypadku pytań lub dużych podatności, zachęcamy do kontaktu bezpośredniego e-mail.

Dziękujemy za dbałość o bezpieczeństwo naszego projektu!
