# Praca inżynierska — Smart Home System

Folder zawiera kompletną pracę inżynierską w formacie Markdown, plan realizacji oraz pliki diagramów PlantUML.

- `praca_inzynierska.md` — pełny dokument pracy inżynierskiej (rozdziały, diagramy, załączniki).
- `plan.md` — szczegółowy, krok-po-kroku plan wykonania i walidacji.
- `diagrams/` — pliki PlantUML dla diagramów UML i architektury.

Pliki są gotowe do wygenerowania PDF (np. `pandoc`) oraz do renderowania diagramów PlantUML.
